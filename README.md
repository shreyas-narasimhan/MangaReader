# MangaReader
A lightweight iOS application built using UIKit that allows users to explore manga titles and browse their chapters using the MangaDex API. The app dynamically fetches manga data, displays chapters with readable language names, and provides seamless navigation to the MangaDex website for an enhanced reading experience.

---
## 🚀 Features

- Browse manga titles and chapters  
- Infinite scrolling for seamless loading  
- Displays chapters with readable language names  
- Smooth navigation and responsive layout  
- Option to open content on the official source  

---

## 🛠️ Tech Stack

- Swift  
- UIKit  
- URLSession  
- Auto Layout
